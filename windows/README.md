# Windows Active Directory Lab

A hands-on Windows Server lab built to simulate a small enterprise environment using Active Directory Domain Services (AD DS). The lab demonstrates domain administration, user and group management, shared folder permissions, and basic troubleshooting.

## Features

- Installed and configured Active Directory Domain Services
- Created and managed Organizational Units (OUs)
- Created domain users and security groups
- Joined a Windows 11 client to the domain
- Configured shared folders with Share and NTFS permissions
- Implemented role-based access control for department resources
- Performed DNS and domain connectivity troubleshooting

## Lab Environment

### Infrastructure

- Windows Server 2025
- Windows 11 Client
- Active Directory Domain Services
- DNS Server

### Configuration

- Static IP addressing
- Domain: `lab.local`
- Organizational Units:
  - IT
  - Finance
- Security Groups:
  - IT
  - Finance
